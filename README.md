
# Clockeye

A simple clock application project.

## Description

A clock project built to display time and practice web development fundamentals.

## Features

- Real-time clock display
- Clean and simple interface
- Responsive design

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Open `index.html` in your browser

## Usage

Open the application to view the current time displayed on the clock.

## Technologies

- HTML
- CSS
- JavaScript
